# Jarvis - Voice-Activated Assistant

Jarvis is a Python-based voice-activated assistant that uses the OpenAI API for natural language processing and SpeechRecognition library for voice input. It can perform various tasks, including opening websites, playing music, providing the current time, and engaging in chat-based conversations.

## Features

- **Voice Recognition:** Utilizes SpeechRecognition library to convert voice input into text for further processing.
- **OpenAI Integration:** Engages with the OpenAI API for chat-based responses and artificial intelligence interactions.
- **Task Automation:** Performs tasks such as opening websites, playing music, and providing the current time.
- **User Interaction:** Responds to user commands and engages in conversations using natural language processing.
